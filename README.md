# PMPToptimizer
'PMPToptimizer' (Post-Modern Portfolio Theory) optimizer provides a set of functions that allows a user to produce optimal portfolios and efficient frontiers with weighted covariance.

## Installation
You can install the development version of FEHreadR from [GitHub](https://github.com/) with the following R command(s):
```r
# Install package "devtools" if not previously installed
# install.packages("devtools")
devtools::install_github("Wesley-Janson/PMPToptimizer")
```
